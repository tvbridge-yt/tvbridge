# TVBridge documentation

This folder collects everything that does not belong in the source tree.

- [architecture.md](./architecture.md) - the moving parts of the broadcast pipeline
- [contributing.md](./contributing.md) - how to propose topics and episodes
- [research-topics/](./research-topics/) - one short brief per topic in the catalog

If you only want to read the catalog programmatically, see the example scripts
under `src/examples/`.
